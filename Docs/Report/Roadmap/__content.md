VR Device와 ZED 카메라를 활용한 AR 당구 구현 청사진

# 기능 지원

기본 구현 목표
- 현재 활성화된 턴의 당구공 오버레이 (주황색 / 흰색 번갈아가면서 하이라이트)
- 큐대 각도에 따라 타격 시 공의 진행 방향 예측, 오버레이

추가 목표
- 당구공 위치 저장, 리플레이 기능
- 게임 상태(점수, 시간 등) 관리

# 구현 목표

- 영상 인식
  - 당구대, 당구공 인식
  - 큐대 인식
- 영상 출력
  - ZED 스테레오 입력 영상 Passthrough to Oculus
  - 3D AR Overlay 출력 ... 엔진 사용 또는 벡터 그래픽으로 직접 렌더링...
