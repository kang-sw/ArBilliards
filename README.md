# AR Billiards

영상 인식을 통해 현실 세계의 당구를 인식하고, 증강 현실로 당구 플레이를 돕습니다. 

# 실행 시 dll 필요

- CUDA v11.0
- OpenCV v4.4.0

# 빌드 의존성

- ZED SDK v3.2.1
- Oculus SDK v1.43.0
- OpenCV 4.4.0
