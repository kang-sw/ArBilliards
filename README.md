# AR Billiards

영상 인식을 통해 현실 세계의 당구를 인식하고, 증강 현실로 당구 플레이를 돕습니다. 

# Tools

- Oculus Rift CV1
- ZED Mobile Stereo Camera
- OpenCV 4.4.0
- Unity 2020.1
